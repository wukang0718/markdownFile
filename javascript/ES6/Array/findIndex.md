# Array.prototype.findIndex

## 作用

`findIndex()`方法返回数组中满足测试函数的第一个元素的**索引**.否则返回-1

## 语法

> ​	arr.findIndex(callback[, thisArg])

### 参数

| 参数          | 作用                                       |
| ------------- | ------------------------------------------ |
| callback      | 针对数组的每个元素都会执行的函数的回调函数 |
| thisArg(可选) |                                            |
