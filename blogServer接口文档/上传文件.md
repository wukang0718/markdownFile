#	后台管理接口

##  上传文件

*   接口说明:

>   编辑页，编辑完markdow文档，上传到服务器并转成html

*   url

>   **upload**

*   参数  

| 字段名称       | 数据类型 | 字段说明         |
| -------------- | -------- | ---------------- |
| fileContent    | string   | markdown文件内容 |
| classification | string   | 分类专栏         |
| label          | string   | 标签             |
*   返回值

```json
{
    success: true,
    data: {},
    message: "上传成功"
}
```
